# Madhunt Lönn Plugins

This mod contains Lönn plugins for [Madhunt](https://gamebanana.com/mods/335146),
allowing mappers to easily create Madhunt maps without being restricted to Ahorn
or having to copy entities from the Ahorn-made maps.

*Madhunt Lönn Plugins* derives from the work of The Madhunt Team, leaded by Popax21.
